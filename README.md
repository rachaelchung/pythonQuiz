# pythonQuiz
This is a very short CLI Quiz App game, specifically made to explore agentic AI programming. Explore the code in quiz.py, but also explore the SPEC.md the AI wrote off of the the REVIEW.md a separate AI looked at. Also take a peek at me REFLECTION.md to learn more about what I learned from this process.

## HOW TO RUN

1. Clone or download this repo
2. Navigate to project folder in terminal

### To run the CLI Loop:
```bash
python3 quiz.py
```
